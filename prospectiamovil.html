Rediseño de CRM en Prospectia SAPI de CV - Aplicación de web a móvil

Prospectia provee a sus clientes una plataforma web tipo CRM, está diseñada específicamente para desarrolladores inmobiliarios.

Sin embargo, no se cuenta con un responsivo adecuado para distintos dispositivos, por lo cual se aplicará las mejores practicas de UI/UX, rediseño Y maquetación del CRM.

Sistema de diseño UI

Se complemento el sistema de diseño web para poder integrar los componentes necesarios para móvil.Se añadio ls siguientes componentes:

  • Tamaño de tipografía: Se busco el tamaño de letra adecuado para una versión móvil.
  • Modal: Diversos modales con secciones fueron realizados para esta implementación.
  • Icon: diferentes iconos universales seran utilizados
  • Botones: Se aplicará los colores, tipografia e iconos para indicar las diferentes acciones

Wireframe en Figma

Se comenzó los distintos wireframe aplicando los componentes establecidos en el sistema de diseño, manteniendo así la coherencia del diseño y la integración del rediseño web al móvil

Los siguientes wireframe son la representación del primer contacto que el usuario tendrá con la app móvil, así como, los datos que debe de estar ingresando.Las líneas azules corresponden a las interacciones del map user journey.
Se utilizo:

  • Sistema de grid: figma maneja esta opción para considerar los espacios que una vez implementado en código debe de considerarse para seguir.
  • Botones, select e input: son los diferentes elementos que el usuario deberá de completar para el ingreso.

Se adecuo la interfaz de "Capacitación" para la versión móvil, quedando de la siguiente manera. Se observa que el acomodo se a modificado para mantener en primer lugar el reproductor de video que es la parte principal. Debajo, se encuentra el listado de videos de capacitación.

Se eliminaron elementos que en versión móvil se puede añadir en otros componentes, solucionando el problema de usabilidad por saturación de información.

Objetivo:

Se realizará una integración de un rediseño a la UI/ UX de Prospectiva acorde a un mapa de sitio elaborado con la metodología de Arquitectura de Información (AI).

Nuestra aplicación móvil Prospectia permitirá a los usuarios gestionar la efectividad de sus clientes, lo que beneficiará controlar sus acciones desde cualquier dispositivo permitiendo hacer un seguimiento correcto y mejor por medio de un viaje de usuario correcto.
  • Desarrollo Frontend (HTML, CSS con Bootstrap y JavaScript).
  • Diseño de prototipos (wireframes y maquetado HTML).

Obstaculos:

La empresa no contaba con una base de sistema de diseño para móvil, el cual, permitira conocer los wireframe de las páginas existentes y su mapas de viaje de usuario.
